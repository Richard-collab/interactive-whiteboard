# Interactive Whiteboard

## Project Documentation

### Overview
This project is designed to provide a collaborative whiteboard experience, where users can draw, write, and share ideas in real-time.

### Features
- Real-time collaboration
- Multiple drawing tools
- User authentication
- Save and load whiteboard sessions

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Richard-collab/interactive-whiteboard.git
   ```
2. Navigate to the project directory:
   ```bash
   cd interactive-whiteboard
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

### Usage

#### Development
To start the development server with Hot Module Replacement (HMR):
```bash
npm run dev
```
This will start the Vite development server at http://localhost:3000.

#### Production Build
To create an optimized production build:
```bash
npm run build
```
The build output will be in the `dist` directory.

#### Preview Production Build
To preview the production build locally:
```bash
npm run preview
```

### Technology Stack
- **React** - UI framework
- **TypeScript** - Type-safe JavaScript
- **Vite** - Build tool with fast HMR
- **Tailwind CSS** - Utility-first CSS framework
- **React Flow** - Node-based graph visualization
